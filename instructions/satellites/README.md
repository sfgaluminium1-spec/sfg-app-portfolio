
# Satellite App Instructions

These instructions are for **satellite applications** in the SFG Aluminium Ecosystem.

## Getting Started

1. Read `self-register.md` to register with the ecosystem
2. Read `mcp-server-template.md` to implement MCP protocol
3. Read `webhook-setup.md` to receive instructions automatically

## Registration Process

All satellite apps must register by:
1. Cloning this repository
2. Creating registration JSON file
3. Backing up codebase
4. Extracting business logic
5. Pushing to GitHub
6. Creating notification issue

## Receiving Instructions

After registration, you'll receive instructions via:
- GitHub webhooks (automatic)
- GitHub issues (manual)
- Direct API calls (orchestrated)

## Support

Questions? Create a GitHub issue with label "question".
