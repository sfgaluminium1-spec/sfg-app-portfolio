# Credit Checker

Automated credit checking system integrated with Xero and credit agencies.

## Features
- Xero customer database sync
- Automated credit checks
- Credit limit management
- Risk assessment
- Alert notifications

## API
Independent API - see `/docs/api-specs/credit-checker-api.yaml`

## Integrations
- Xero API
- Credit reference agencies
- NEXUS customer database

## Tier Access
- Tier 1: Full access
- Tier 2: Finance team access
