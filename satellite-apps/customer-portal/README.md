# Customer Portal

Customer-facing web application for order tracking, quotes, and communication with SFG Aluminium.

## Features
- Order tracking and status updates
- Quote requests and approvals
- Document uploads and downloads
- Communication portal
- Invoice viewing and payment

## API
Independent API - see `/docs/api-specs/customer-portal-api.yaml`

## Tier Access
Available to all customer tiers with appropriate credit limits.
