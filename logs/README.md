
# Logs Directory

This directory stores application logs and audit trails.

## Log Types

- **app.log** - General application logs
- **webhook.log** - Webhook events and responses
- **api.log** - API endpoint access logs
- **error.log** - Error and exception logs
- **audit.log** - Security and compliance audit trail

## Log Rotation

Logs are rotated daily and archived for 30 days.

## Monitoring

Logs are monitored for:
- Error rates
- Response times
- Security events
- Integration failures
