# Manus Communication Files

Communication protocols and files specifically for the Manus autonomous agent.

## Purpose
Coordinate tasks, status updates, and handoffs with the Manus agent.

## File Types
- **Task Definitions**: Structured task requests for Manus
- **Status Updates**: Progress reports from Manus
- **Handoff Protocols**: Procedures for human-agent handoffs
- **Error Reports**: Structured error and exception reporting

## Communication Format
All Manus communication follows structured JSON format for machine readability and human oversight.
