# Documentation

Comprehensive documentation for the SFG Aluminium ecosystem.

## Directory Structure

### autonomous-instructions/
Instructions and guidelines for autonomous AI agents working within the ecosystem.
- System architecture explanations
- Task execution guidelines
- Safety and compliance rules
- Integration patterns

### manus-communication/
Communication files specifically for the Manus autonomous agent.
- Task definitions
- Status updates
- Coordination protocols
- Handoff procedures

### api-specs/
API specifications for each application in the ecosystem.
- OpenAPI/Swagger definitions
- Authentication requirements
- Endpoint documentation
- Request/response examples

### privacy-policy/
Privacy policy documents covering:
- Data collection and usage
- GDPR compliance
- Customer data protection
- Staff data handling
- 7-year retention policy

### terms-of-service/
Terms of service documents for:
- Customer portal usage
- Staff system access
- API usage terms
- Service level agreements

## Documentation Standards
- All API specs follow OpenAPI 3.0 standard
- Privacy and legal documents reviewed quarterly
- Agent instructions updated with each system change
- Version control for all documentation

## Contributing
When adding new features or applications:
1. Update relevant API specs
2. Add autonomous agent instructions if needed
3. Review privacy implications
4. Update terms of service if customer-facing
