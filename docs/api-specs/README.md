# API Specifications

OpenAPI/Swagger specifications for all APIs in the SFG Aluminium ecosystem.

## Purpose
Provide complete, standardized API documentation for all applications.

## Contents
- NEXUS API specification
- Satellite app API specifications
- Webhook API documentation
- Authentication and authorization specs

## Standards
All API specs follow OpenAPI 3.0 standard and include:
- Endpoint definitions
- Request/response schemas
- Authentication requirements
- Error codes and handling
- Rate limiting information
- Example requests and responses
