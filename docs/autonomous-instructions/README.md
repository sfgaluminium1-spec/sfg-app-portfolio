# Autonomous Agent Instructions

Guidelines and instructions for autonomous AI agents operating within the SFG Aluminium ecosystem.

## Purpose
Provide clear, structured instructions for AI agents to safely and effectively perform tasks within the system.

## Contents
- System architecture overview
- Task execution patterns
- Safety constraints and guardrails
- Integration protocols
- Error handling procedures
- Escalation paths

## Agent Types
- **Manus**: Primary autonomous agent for system orchestration
- **Satellite Agents**: Specialized agents for individual satellite apps
- **Support Agents**: Customer service and internal support agents

## Safety Guidelines
All autonomous agents must:
1. Respect tier-based access controls
2. Never exceed spending limits
3. Log all actions for audit
4. Escalate uncertain decisions
5. Follow UK compliance requirements
