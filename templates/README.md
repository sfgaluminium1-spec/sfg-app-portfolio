
# Templates Directory

This directory contains standardized templates for the SFG NEXUS ecosystem.

## Available Templates

### app-registration-template.json
Standard format for app registration metadata including:
- Technical specifications
- Business metrics
- Integration details
- Health status

### README-template.md
Template for application documentation including:
- Overview and purpose
- Installation instructions
- Configuration guide
- API documentation
- Troubleshooting

### business-logic-template.json
Template for extracting business logic including:
- Business rules
- Workflows
- Calculations
- Decision trees

## Usage

1. Copy the appropriate template
2. Fill in your application-specific details
3. Validate JSON format (for JSON templates)
4. Commit to the repository

## Validation

All templates include:
- Required fields marked with `[REQUIRED]`
- Optional fields marked with `[OPTIONAL]`
- Example values for guidance
- Format specifications
